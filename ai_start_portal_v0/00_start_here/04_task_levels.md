# Три уровня заданий

В курсе есть три уровня погружения.

Это не оценки и не соревнование.

Это разные режимы владения AI-работой.

## A1: использовать workflow

Для кого:

- новичок;
- корреспондент в дедлайне;
- человек, который не хочет становиться "AI-специалистом";
- участник, которому нужно быстро получить пользу.

Что делать:

```text
готовый workflow
-> исходный материал
-> AI draft
-> factual check
-> сохранить пример
```

Результат:

- черновик;
- список фактов;
- вопросы на проверку;
- сохраненный пример.

## A2: адаптировать output

Для кого:

- редактор;
- SMM;
- корреспондент, который уже освоил A1;
- человек, которому нужно несколько форматов.

Что делать:

```text
output A1
-> версия для другого канала
-> сравнение
-> правка тона / структуры
```

Результат:

- сайт + Telegram;
- headlines;
- social post;
- quote card text;
- clip ideas.

## A3: улучшать систему

Для кого:

- process owner;
- редактор;
- сильный участник;
- будущий AI champion команды.

Что делать:

```text
3 похожие проблемы
-> update proposal
-> тест на примерах
-> weekly review
-> changelog
```

Результат:

- улучшенный workflow;
- новый quality gate;
- обновленный context;
- новая known failure.

## Главное правило

```text
A1 = достаточно
A2 = глубже
A3 = владение системой
```

Если вы сделали A1 качественно, вы уже используете AI Start правильно.
