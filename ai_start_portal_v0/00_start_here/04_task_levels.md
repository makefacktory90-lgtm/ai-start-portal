# Что уносим после занятия

## Base

Минимальный результат после занятия:

- сформулировать рабочую AI-задачу;
- отделить источник, формат, ограничения и проверку;
- получить проверяемый черновик;
- отметить, что должен проверить человек;
- сохранить пример в hub.

Результат:

```text
задача
-> источник / материал
-> AI draft
-> human check
-> сохраненный пример
```

## Opti

Дополнительный уровень для своей роли:

- адаптировать результат под канал или формат;
- собрать 2-3 версии из одного материала;
- сравнить AI-маршрут с ручным способом;
- оформить личный рабочий shortcut.

Результат:

```text
один материал
-> 2-3 рабочих результата
-> проверка
-> личный shortcut
```

## Advanced

Уровень для owners и участников, которые будут собирать систему:

- описать повторяющуюся задачу;
- определить вход, выход, владельца и проверку;
- выбрать инструменты под задачу;
- предложить обновление prompt, context или gate.

Результат:

```text
повторяющаяся задача
-> workflow
-> набор инструментов
-> quality gate
-> update proposal
```

## Scout

Дополнительный обзор рынка:

- какие инструменты стоит попробовать;
- где есть риск для данных, фактов или репутации;
- кто в редакции может тестировать инструмент;
- какой результат нужно сохранить в hub.

Результат:

```text
карта инструментов
-> что попробовать
-> где риск
-> кто тестирует
```
