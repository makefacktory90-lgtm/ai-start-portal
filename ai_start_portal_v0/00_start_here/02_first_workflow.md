# Первый workflow

Начни отсюда:

```text
02_workflows/01_note_to_news_draft/workflow.md
```

Он подходит, если у тебя есть:

- заметка с места события;
- короткая расшифровка;
- пресс-релиз;
- сообщение от коллеги;
- набросок новости;
- сырой материал после звонка / интервью.

## Что получится

```text
1. Факты из материала
2. Что нужно проверить
3. Черновик новости
4. Telegram-версия
5. Риски / вопросы редактору
```

## Главное правило

Не проси AI "сделать красиво" до того, как он выделил факты и вопросы.

Сначала факты. Потом текст.
