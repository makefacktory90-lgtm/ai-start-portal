# Прочитайте сначала

Это рабочая папка курса AI Start.

Здесь лежат:

- презентации;
- практические материалы;
- prompts;
- quality gates;
- домашние задания;
- записи и summary после встреч.

## Быстрый старт

1. Откройте портал.
2. Перейдите к первой лекции.
3. Откройте `safe source Deck 01`.
4. Выполните research-прогон по готовому prompt.
5. Сохраните результат и вопросы к следующему занятию.

## Что сохраняем после практики

```text
source
-> prompt
-> output
-> human gate
-> hub note
```

## Правила работы

- Не загружайте во внешние AI-сервисы персональные данные, договоры, доступы, непубличные документы и чувствительные материалы.
- Если материал нельзя использовать, возьмите synthetic source или обезличенный фрагмент.
- Факты, которых нет в источнике, помечайте как `нужно проверить`.
- Перед публикацией используйте human gate.

## Уровни участия

### Base

Выполнить готовый workflow и проверить результат.

### Opti

Адаптировать workflow под свой канал, роль или формат.

### Advanced

Предложить обновление prompt, gate или workflow на основе рабочего примера.
