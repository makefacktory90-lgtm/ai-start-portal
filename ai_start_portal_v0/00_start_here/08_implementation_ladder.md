# Что внедряем после уроков

После каждого занятия остается один небольшой рабочий результат.

Формат:

```text
один материал
-> один AI-шаг
-> одна проверка
-> один сохраненный пример
```

## После Lesson 01. Start map

Результат:

- Codex playground;
- research prompt;
- source map;
- human gate.

Время: 20-30 минут.

## После Lesson 02. Text OS

Результат:

- Source Map;
- черновик для одного канала;
- factual gate;
- список правок перед публикацией.

Время: 30-40 минут.

## После Lesson 03. Video acceleration

Результат:

- транскрипт или описание видео с таймкодами;
- 5-7 clip candidates;
- top-3 для производства;
- video gate notes.

Время: 30-60 минут.

## После Lesson 04. Visual conveyor

Результат:

- visual brief;
- структура карточек / обложки;
- 3 prompt options;
- visual gate.

Время: 30-45 минут.

## После Lesson 05. Audio OS

Результат:

- spoken script на 60-90 секунд;
- read-aloud notes;
- audio gate update.

Время: 25-35 минут.

## После Lesson 06. Editorial contour

Результат:

- 10-20 сигналов;
- weekly digest;
- content grid;
- source risk notes.

Время: 35-40 минут.

## После Lesson 07. Hub build

Результат:

- один удачный AI-output;
- один слабый или рискованный AI-output;
- одно правило для hub.

Время: 30 минут.

## После Lesson 08. Agents

Результат:

- одна повторяющаяся задача;
- manual / semi-auto / future agent version;
- source, tools, memory, human gate.

Время: 30-40 минут.

## После Lesson 09. Practice clinic

Результат:

- один кейс;
- что сломалось: prompt, context, gate, tool или owner;
- update proposal.

Время: 30 минут.

## После Lesson 10. Final kit

Результат:

- owners;
- 3 weekly actions;
- 30-day dashboard;
- дата первого review.

Время: 40 минут.

## Вопросы в чате

Пишите вопрос по шаблону:

```text
1. Что хотели сделать:
2. Где остановились:
3. Инструмент:
4. Скрин / текст ошибки:
5. Что уже пробовали:
```

Для разбора материала:

```text
1. Source:
2. Что нужно получить:
3. Где будет опубликовано:
4. Что нельзя потерять:
5. Какой риск:
```
