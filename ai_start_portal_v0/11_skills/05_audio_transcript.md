# 05 Audio / Transcript

## Назначение

Сделать из текста или видео транскрипт, spoken script, аудиоверсию и проверку звучания.

## Когда использовать

- нужно расшифровать эфир, интервью, лекцию или видео;
- письменный материал нужно адаптировать в звучащий текст;
- нужен фрагмент для подкаста, аудиоанонса или озвучки.

## Что нужно до старта

- исходное аудио / видео / текст;
- Whisper / MacWhisper / Superwhisper или доступная транскрибация;
- правила произношения имен, географии, компаний;
- audio quality gate.

## Основной prompt

```text
Ты редактор звучащего текста.

На основе source сделай spoken script.
Не добавляй фактов.
Сохрани смысл и осторожность.

Сделай:
1. Главный смысл.
2. Обязательные факты.
3. Intro hook на 1-2 фразы.
4. Spoken script на 60-90 секунд.
5. Места, которые надо произнести особенно аккуратно.
6. Слова / имена / названия для проверки.
7. Audio QA: что проверить перед записью или публикацией.

Source:
[вставьте текст или транскрипт]
```

## Output format

```text
## Главный смысл
## Обязательные факты
## Intro hook
## Spoken script
## Pronunciation notes
## Audio QA
```

## Human gate

- текст можно произнести вслух;
- имена, должности, география и числа проверены;
- нет добавленных фактов;
- ритм не звучит как письменная заметка;
- ответственный человек слушает финальный фрагмент.

## Домашка Base / Opti / Advanced

- Base: сделать spoken script на 60-90 секунд.
- Opti: сделать две версии: нейтральную и более разговорную.
- Advanced: обновить audio workflow, если script звучит искусственно.
