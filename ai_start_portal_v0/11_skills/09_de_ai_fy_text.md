# 09 De-AI-fy Text

Status: draft for lesson 02. Based on Ira's draft `de-ai-fy-text.md` and humanize notes. Do not publish before 15 May.

## Назначение

Убрать из текста AI-гладкость: пустые обобщения, рекламный тон, одинаковый ритм, фальшивую человечность и уверенные утверждения без источника.

## Когда использовать

- текст выглядит формально корректным, но не звучит живо;
- черновик похож на пресс-релиз без автора;
- AI добавил масштаб, которого нет в источнике;
- нужно показать участникам, почему "сделай человечнее" - плохая команда;
- перед publication package нужен smell test.

## Основной prompt

```text
Проверь текст на AI-гладкость.

Не переписывай сразу.

Найди:
1. Общие фразы без факта.
2. Канцелярит или рекламный тон.
3. Симметричный ритм.
4. Пустые усилители масштаба.
5. Утверждения без источника.
6. Фразы, которые звучат слишком готово.
7. Места, где текст объясняет то, что должен показать факт.

Для каждого места дай:
- цитату;
- почему это пластик;
- чем заменить: факт / цитата / место / герой / число / конкретное последствие;
- нужен ли human check.

Потом предложи версию не длиннее исходной.
Новые факты не добавляй.
```

## Parallel critic mode

Если текст важный, запустить 4 критика:

1. Generic detector: общие фразы, пустые усилители.
2. Rhythm analyzer: одинаковая длина фраз, одинаковые абзацы.
3. Specificity checker: где нужен факт, цитата, место, число, герой.
4. Fact checker: какие утверждения нужно проверить.

## Rewrite rules

- итоговая версия не длиннее исходника;
- заменить абстракцию на проверяемую конкретику;
- не добавлять "человечность" через фальшивые эмоции;
- разрешить живой ритм: коротко, потом длиннее, потом пауза;
- убрать финальные драматические бантики;
- не трогать полезные списки, команды, технические фрагменты.

## Human gate

- все новые детали должны быть в источнике;
- если факт не подтвержден, он уходит в "нужно проверить";
- автор/редактор решает, где допустима ирония или разговорность;
- хороший результат может быть менее гладким, но более точным и редакционно убедительным.

## Домашка Base / Opti / Advanced

- Base: найти 5 пластиковых мест в AI-черновике.
- Opti: переписать текст через fact / quote / place / person / number.
- Advanced: сохранить rejected example и правило для Style Graph.
