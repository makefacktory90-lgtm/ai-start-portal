# 03 Video Clips

## Назначение

Превратить длинное видео, сюжет, эфир или транскрипт в кандидатов на вертикальные клипы с хуками, субтитрами и монтажным gate.

## Когда использовать

- есть длинный сюжет или интервью;
- нужно быстро понять, есть ли материал для вертикалок;
- нужен publication package для SMM / видео-команды.

## Что нужно до старта

- видео, транскрипт или монтажный план с таймкодами;
- Whisper / MacWhisper / Superwhisper или доступная расшифровка;
- при необходимости OpusClip / Gling / Descript / текущий NLE;
- редакционный запрет: не вырывать цитаты из контекста.

## Основной prompt

```text
Ты ассистент видеоредактора.

Материал: [транскрипт / таймкоды / описание видео].
Цель публикации: [информировать / объяснить / вызвать обсуждение / анонсировать].
Аудитория: [кто смотрит].

Запреты:
- не вырывать фразу из контекста;
- не усиливать конфликт искусственно;
- не обещать больше, чем есть в видео;
- не придумывать цитаты и факты.

Сделай:
1. Карту видео по смысловым блокам с таймкодами.
2. 7 кандидатов на вертикальный клип.
3. Для каждого: таймкод, смысл, hook, обложка, описание, риск.
4. Выбери top-3 для производства.
5. Для top-3 сделай subtitle chunks короткими строками.
6. Отдельно напиши, что проверяет редактор.
7. Отдельно напиши, что проверяет монтажер.
```

## Output format

```text
## Карта видео
## 7 кандидатов
## Top-3
## Hook / subtitle chunks / cover text
## Риски редактора
## Риски монтажа
```

## Human gate

- клип не меняет смысл героя;
- hook не обещает больше, чем есть в видео;
- субтитры не ломают цитату;
- есть права и разрешения;
- качество звука и картинки пригодно;
- монтажер подтверждает, что фрагмент можно собрать.
