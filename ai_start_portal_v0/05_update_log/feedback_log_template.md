# Feedback Log Template

Используй, когда workflow сработал плохо или требует улучшения.

## Запись

```text
Date:
Team:
Participant:
Workflow:
Material type:

What AI produced:

What was wrong:

What human changed:

Desired behavior next time:

Severity:
low / medium / high

Update target:
context / workflow / quality gate / examples / personal note

Evidence:
link or pasted excerpt

Suggested owner:

Status:
new / grouped / testing / approved / rejected / personal
```

## Правило

Одна странная ошибка - это пример.

Три похожие ошибки - это кандидат на update.
