# Weekly Review Template

Для process owners / champions.

Проводить раз в неделю.

## Inputs

Собрать:

- approved examples;
- rejected examples;
- feedback log entries;
- repeated corrections;
- missing context;
- high-risk outputs.

## Review Questions

```text
1. Какие проблемы повторились 3+ раза?
2. Это проблема context, workflow, quality gate или personal note?
3. Есть ли evidence?
4. Какие примеры тестируем?
5. Обновление безопасно для всей команды?
6. Кто approves?
7. Что будет rollback version?
```

## Decision

```text
Status:
approved / testing / rejected / personal

Changed files:

Reason:

Examples tested:

Owner:

Next review date:
```

## Важно

Не превращать weekly review в общую болталку про AI.

Цель:

```text
problem -> evidence -> update decision
```
