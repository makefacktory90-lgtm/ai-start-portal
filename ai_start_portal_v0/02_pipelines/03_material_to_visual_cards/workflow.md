Главный результат: материал превращается в визуальный бриф, карточки, обложку, простую инфографику или AI-asset без потери фактов.

## Когда использовать

Используй, если нужно подготовить:

- карточки;
- обложку;
- цитатную карточку;
- carousel outline;
- простую инфографику;
- AI-иллюстрацию как фон / образ;
- бриф дизайнеру или в Figma / Canva.

## Главное правило

Не просим image model писать финальный текст с фактами.

Факты, числа, имена и цитаты должны жить в редактируемом шаблоне и проходить human review. AI-изображение может быть образом, фоном или иллюстративным asset, но не источником публикационного текста.

## Стартовая команда

```text
Открой AI Start Kit.
Используй Pipeline 03: материал -> визуал и карточки.

Задача:
1. Извлеки главный смысл материала.
2. Отдельно выпиши факты, которые нельзя искажать.
3. Предложи 3 визуальных формата:
   - обложка;
   - 3-5 карточек;
   - простая инфографика / quote card.
4. Для выбранного формата сделай:
   - структуру карточек;
   - текст карточек;
   - короткие подписи;
   - visual brief;
   - prompts для AI-asset, если уместно.
5. Не выдумывай факты, числа, цитаты и визуальные детали, которых нет в материале.
6. Все сомнительное вынеси в блок "нужно проверить".

Материал:
[вставьте ниже]
```

## A1: подготовить текст для визуального формата

Сделай:

```text
главный смысл -> факты -> 3 формата -> структура карточек -> риски
```

## A2: собрать visual brief

После A1 попроси:

```text
Собери brief для дизайнера / Figma / Canva:
- формат;
- размеры, если известны;
- структура;
- тексты;
- визуальный стиль;
- что должно быть редактируемым;
- что нельзя генерировать картинкой;
- checklist перед публикацией.
```

## A3: обновить visual gate

Если визуальный output слабый, попроси:

```text
Сделай update proposal для visual quality gate.
Укажи:
- где AI ломает смысл;
- где текст становится нечитаемым;
- где изображение создает ложный факт;
- какое правило добавить;
- как проверить на 3 примерах.
```

## Visual quality gate

Проверить:

- текст читается на мобильном;
- факты не зашиты в нер редактируемую картинку;
- цитаты точные;
- числа и даты проверены;
- визуал не создает ложное событие;
- AI-asset не выглядит как документальный факт, если это иллюстрация;
- бренд / шаблон не сломан;
- карточки не перегружены.
