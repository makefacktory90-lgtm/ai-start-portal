# Tool radar: video / visual / agentic editing

Статус: рабочий список для усиления Deck 03 / 04 / 08.  
Цель: показывать редакциям, что уже есть в мире, но не продавать это как готовую замену профессии.

## Главный тезис

Профи сегодня часто сделает руками быстрее и качественнее.

Но рынок движется туда, где AI забирает:

- первый rough cut;
- поиск фрагментов;
- удаление пауз и мусорных дублей;
- субтитры и reframe;
- первичную визуальную генерацию;
- timeline notes;
- экспорт в монтажную систему.

Задача редакции: не поверить в кнопку "сделай хорошо", а научиться тестировать, что уже ускоряет конкретный участок работы.

## Video acceleration

### Opus Clip

Что показывать:

- long-form video -> shorts;
- auto clipping;
- captions;
- layout / reframe;
- remove fillers and pauses;
- AI B-roll / overlay features.

Источник:

- https://help.opus.pro/docs/article/clipanything-qa-3

### Gling

Что показывать:

- remove silences;
- remove filler words;
- remove bad takes;
- text-based trimmer;
- speech enhancement;
- XML export to Premiere / Final Cut / DaVinci Resolve.

Источник:

- https://www.gling.ai/desktop

### Descript

Что показывать:

- transcript-based editing;
- Studio Sound;
- filler word removal;
- shorten word gaps;
- notes for publishing.

Источник:

- https://help.descript.com/hc/en-us/articles/27252457732237-Meet-Underlord-your-AI-sidekick

### AutoPod

Что показывать:

- Premiere Pro plug-ins for video podcasts / shows;
- Multi-Camera Editor;
- Jump Cut Editor;
- social clips.

Источник:

- https://www.autopod.fm/

## Agentic / timeline editing

### ButterCut

Что показывать:

- Claude Code analyzes footage;
- builds audio / visual transcripts;
- user describes narrative, duration, pacing;
- exports a timeline for Final Cut, Premiere or DaVinci Resolve;
- editor refines manually.

Источник:

- https://buttercut.io/

### Premiere Pro MCP

Что показывать осторожно:

- MCP-compatible clients can control Premiere through a server / panel;
- timeline editing, captions, effects, export;
- это experimental / technical stack, не "включили и всем раздали".

Источники:

- https://conare.ai/marketplace/mcp/adobe-premiere-pro
- https://mcpmarket.com/server/adobe-premiere-pro-1

## Visual conveyor

Что показывать:

- reference -> style rules;
- style rules -> prompt project;
- prompt project -> Higgsfield / Kling / Runway generations;
- generated asset -> animation;
- final facts and text -> Figma / Canva / Premiere editable layer.

Правило:

```text
AI gives image / motion asset.
Template keeps text, facts, captions and layout.
Human gate owns publication.
```

## Как говорить на лекции

Не говорить:

> "Теперь нейронки всё монтируют".

Говорить:

> "Смотрите, вот какие куски вашего таймлайна уже начинают автоматизироваться. Где-то вам руками быстрее. Окей. Но если задача - выйти на другой уровень скорости, вы должны знать, что тестировать и как сравнивать".

## Что дать участникам

Таблица для теста:

```text
инструмент
-> какую рутину забирает
-> вход
-> выход
-> где качество хуже человека
-> что должен проверить профи
-> стоит ли тестировать в редакции
```
