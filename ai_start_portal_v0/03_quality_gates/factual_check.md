# Quality Gate: factual check

Используй перед тем, как показывать AI output редактору или публиковать дальше.

## Быстрая проверка

Ответь:

```text
1. Все факты есть в исходном материале?
2. Все имена, должности, даты и числа проверены?
3. Есть ли цитаты? Если да, есть ли источник?
4. AI сделал выводы, которых нет в материале?
5. Есть ли блок "нужно проверить"?
6. Может ли заголовок исказить смысл?
7. Есть ли репутационный, юридический или человеческий риск?
```

## Команда для проверки

```text
Проверь этот текст по factual_check из AI Start Kit.
Не переписывай сразу.
Сначала сделай таблицу:
- утверждение;
- есть ли это в исходном материале;
- риск;
- что нужно проверить.

Потом предложи осторожную исправленную версию.

Исходный материал:
[вставьте источник]

AI output:
[вставьте текст]
```

## Красные флаги

- "по словам экспертов" без экспертов;
- точные числа без источника;
- причинно-следственные связи без подтверждения;
- цитаты без расшифровки / записи;
- оценочные слова в новости;
- уверенный тон там, где есть неопределенность;
- обобщения про жителей, работников, компанию, город.

## Решение

Если риск низкий:

- исправить текст;
- сохранить пример при необходимости.

Если риск средний:

- показать редактору;
- добавить вопрос в `нужно проверить`.

Если риск высокий:

- не публиковать;
- отправить редактору / ответственному;
- сохранить rejected example.
