# Editorial contour quality gate

Статус: draft v0.1  
Связано с: Pipeline 05, блок 22 мая и финальные встречи.

## Проверка digest / content grid

- Источник каждого сигнала указан.
- Сомнительный источник помечен.
- Идея материала не выдана за факт.
- Важность темы объяснена, а не просто проставлена.
- Контент-сетка реалистична по людям и времени.
- У каждой позиции есть owner или next step.
- Для каждой позиции указан pipeline.
- Есть блок "что проверить".
- Weekly update основан на evidence, а не на ощущении.

## Красные флаги

- AI смешал проверенные факты и слухи.
- Контент-сетка выглядит красиво, но ее некому делать.
- Telegram-регулярность заменяет редакционный смысл.
- Update proposal не указывает, какой файл менять.
- Автоматизация предлагается до накопления examples.

## Что сохранять в hub

- weekly digest;
- content grid;
- source risk notes;
- owner decisions;
- update proposals;
- rejected automation candidates.
