# Visual quality gate

Статус: draft v0.1  
Связано с: Pipeline 03, блок 19 мая.

## Проверка перед публикацией

- Главный смысл понятен за 3-5 секунд.
- Текст читается на мобильном.
- Факты, числа, имена и цитаты остаются редактируемым слоем.
- AI-asset не выглядит как документальный факт, если это иллюстрация.
- Визуал не создает события, которого не было.
- Обложка / карточка не обещает больше, чем материал дает.
- Формат подходит каналу: cover, title, intertitle, quote, card, interactive quad.
- Есть место под бренд / шаблон / служебные элементы.
- Есть список фактов, которые нельзя искажать.

## Красные флаги

- Финальный текст запечен в картинку.
- Карточка красивая, но непроверяемая.
- Слишком много текста для одного экрана.
- AI дорисовал людей, места, документы или объекты как будто это факт.
- Формат выглядит как разовый дизайн, который команда не сможет повторять.

## Что сохранять в hub

- visual brief;
- структуру карточек;
- хорошие / плохие примеры;
- asset prompt;
- правило, какой текст нельзя отдавать image model.
