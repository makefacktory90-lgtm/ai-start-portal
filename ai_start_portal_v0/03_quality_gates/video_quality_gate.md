# Video quality gate

Статус: draft v0.1  
Связано с: Pipeline 02, блок 19 мая.

## Проверка перед производством

- Фрагмент понятен без критичного внешнего контекста.
- Hook не обещает больше, чем есть в видео.
- Цитата не искажена.
- Нет обрезки, которая меняет позицию героя.
- Есть монтажная возможность: кадр, звук, паузы, начало и конец.
- Subtitle chunks короткие и читаемые.
- Cover text не кликбейтит и не добавляет новый факт.
- Есть права / разрешение на публикацию.
- Редактор понимает, зачем этот клип нужен аудитории.

## Красные флаги

- "Звучит мощно", но смысл становится другим.
- Фрагмент держится только на эмоциональной фразе без факта.
- Первый кадр не работает как начало.
- AI выбрал кусок, который требует длинного объяснения.
- В subtitles появились слова, которых нет в исходнике.

## Что сохранять в hub

- таблицу кандидатов;
- топ-3 с причинами выбора;
- отклоненные клипы с причиной;
- удачные subtitle chunks;
- правила, которые надо добавить в prompt.
