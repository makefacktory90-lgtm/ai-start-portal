Что купить / подключить к старту

Для первых занятий нужен короткий набор: основной LLM-доступ, agent layer для рабочей папки и проверенный маршрут транскрибации. Production-сервисы подключаем пилотно, когда понятен конкретный сценарий.

01 LLM layer ChatGPT / Claude / Gemini / согласованный корпоративный аналог
02 Agent layer Codex / Claude Code для owners, ведущей и advanced-уровня
03 Транскрибация Whisper / MacWhisper / Wispr Flow / текущий route

// слои

Как объясняем закупку

LLM layer

ChatGPT / Claude / Gemini

Помогают думать, писать, разбирать источники, собирать варианты и видеть слабые места.

Agent layer

Codex / Claude Code

Ведут рабочую папку: prompts, gates, examples, source maps, outputs и обновления hub.

Production layer

Higgsfield / OpusClip / Submagic / Gling

Делают конкретные операции. Покупаем пилотно, не всем участникам сразу.

// минимум

Минимум для первых двух лекций

A

Основной LLM-доступ

Нужен для research, анализа источников, Style Graph, tone of voice, smell test и текстовых черновиков. Подходит ChatGPT, Claude, Gemini или согласованный корпоративный аналог.

ChatGPT
B

Agent layer

Это отдельный слой для работы с папкой, prompts, gates и hub. Нужен Сергею, ведущей и тем, кто идет в advanced. Участники ставят самостоятельно по записи.

Codex CLI
C

Транскрибация

Нужна для видео, интервью, эфиров и записей занятий. Можно использовать текущий approved route, Whisper-стек или Wispr Flow для голосового ввода и быстрых заметок.

Whisper
D

Telegram-чат

Нужен для вопросов по установке, домашке и первым outputs. Это не закупка, но обязательная часть поддержки курса.

// опционально

Что добавляем только при необходимости

Perplexity Опционально: быстрый research без кода и API. Можно обойтись основным AI-доступом и обычным search-подходом. TUN VPN / WhyNot VPN Опционально: использовать маршрут доступа, который дает Ира, если он согласован с участниками. Wispr Flow Опционально: голосовой ввод, быстрые заметки и черновики. Запросить промокод на 1 месяц. Exa Search API для researcher-сценариев; достаточно 1 technical owner на пилот. Tavily Альтернатива Exa для agent/search-сценариев; выбираем один основной маршрут. Higgsfield 1-2 места для визуального блока и демонстраций, не для всех участников. OpusClip Пилот для длинное видео → короткие клипы после решения с видео-специалистом. Gling Пилот для rough cut, пауз и подготовки монтажного черновика. Submagic Пилот для субтитров и упаковки коротких видео, если подтвердит видео-команда.
// решение

Рекомендуемое решение на старт

1

Проверяем сейчас

Основной LLM-доступ, Telegram-чат, транскрибация, Codex / Claude Code для команды курса и advanced.

2

Опции

Perplexity, VPN и Wispr Flow. Эти инструменты не обязательны для всех участников.

3

Пилоты

Exa или Tavily для technical owner, Higgsfield для visual demo, один video acceleration сервис после консультации с видео-специалистом.

4

Не покупаем массово

Несколько video-сервисов одновременно, клонирование голоса, автоматизацию публикаций, отдельные доступы всем участникам без понятного сценария применения.

// коротко

Промокод

По Wispr Flow: запросить промокод на 1 месяц. Я не указываю код на странице, пока он не подтвержден, чтобы участники не получили нерабочую ссылку.
