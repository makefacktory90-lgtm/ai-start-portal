// PROMPT ANATOMY

Промпт описывает задачу, источник, формат и проверку.

Задача: что получить и для какого канала.

Источник: какие факты, цитаты и ограничения есть.

Роль: кто проверяет и по каким критериям.

Формат: структура output, длина, варианты.

Gate: что обязательно проверить перед выпуском.

// STARTER PROMPT

Первый промпт не пишет финал.
Он собирает редакционную карту.

Возьми материал ниже и разложи его для редакционной работы.

Не пиши финальную публикацию. Сначала сделай карту.

1. Выдели проверяемые факты, действующих лиц, даты, места и цитаты.
2. Отметь, чего не хватает для уверенной публикации.
3. Предложи 3 формата: сайт, Telegram, короткий social-анонс.
4. Для каждого формата укажи риск: факт, тон, контекст, юридическая/репутационная чувствительность.
5. Сформулируй human gate: что человек обязан проверить перед выпуском.
6. Ничего не додумывай. Если данных нет, напиши: "нет в источнике".
// ВОЗРАЖЕНИЕ 01 · КЛИШЕ

Нейротекст видно не по факту AI, а по пустой гладкости

симметрияодинаковые абзацы, ровные списки, слишком уверенный тон

ложная полнотатекст звучит готовым, но не хватает фактов

// SMELL TEST

Ловим пустые формулировки и возвращаем текст к источнику.

04 мин. Найти 3 фразы, которые можно вставить почти в любой материал.

05 мин. Заменить каждую фразу на факт, цитату, место, героя или число.

03 мин. Записать правило: какой smell больше не пропускаем.

03 мин. Два ответа разбираем вслух и превращаем в gate.

// ДОМАШНЕЕ ЗАКРЕПЛЕНИЕ

После первого занятия каждый делает один маленький прогон.

1. Выбрать один реальный материал, который можно безопасно использовать.

2. Прогнать starter prompt: source map, risks, formats, human gate.

3. Сохранить output и пометить: что помогло, что было опасно, что нужно уточнить.

4. Принести на следующий раз один пример: good output или fail output.
