Первый рабочий маршрут:
материал становится системой форматов

source EXO context ToV Style Graph Repurpose Boss QA hub update

Не ищем “магический промпт”. Разбираем маршрут: как извлечь смысл, удержать голос, пересобрать материал в каналы и сохранить удачный прием для команды.

// КАК УСТРОЕН ПРОМПТ

Промпт — это упаковка маршрута

Задача: что нужно получить и для какого канала.

Источник: какие факты, цитаты и ограничения есть в материале.

Роль: кто проверяет результат и по каким критериям.

Формат: структура output, длина, тон, варианты.

Gate: что обязательно проверить перед выпуском.

// STARTER PROMPT

Первый промпт не пишет финал.
Он собирает карту материала.

Возьмите материал ниже и разберите его для редакционной работы.

1. Выделите факты, цитаты, действующих лиц и даты.
2. Отметьте, чего не хватает для уверенной публикации.
3. Предложите 3 возможных формата: сайт, Telegram, короткий social.
4. Для каждого формата укажите риск и human gate.
5. Ничего не додумывайте, если этого нет в источнике.

Нейротекст видно не по факту AI, а по пустой гладкости

ложная полнотатекст звучит готовым, но в нем не хватает фактов и проверок

Безопасность — это не обещание сервиса, а правило редакции
