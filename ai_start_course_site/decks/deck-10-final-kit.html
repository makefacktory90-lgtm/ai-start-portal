// REVIEW PROMPT

Финальный промпт для ревью комплекта

Проверь AI Start Kit как рабочую систему редакции.

Оцени:
1. Есть ли понятный вход для новичка.
2. Есть ли 5 рабочих pipelines.
3. Есть ли quality gates.
4. Есть ли approved / rejected examples.
5. Есть ли owners и weekly review.
6. Что непонятно без автора курса.
7. Что надо упростить до первого самостоятельного применения.
// 30 DAYS

30-дневный цикл применения

week 1

Один текстовый package и один saved example.

week 2

Один video / visual test и один rejected example.

week 3

Один editorial contour digest и content grid.

week 4

Review: keep, revise, kill, automate later.

// CLOSE

Финальная фраза: AI не забирает профессию. Он поднимает требования к постановке задачи.

Кто умеет формулировать задачу, проверять результат и собирать систему, тот ускоряется. Кто ждет кнопку "сделай хорошо", тот получает красивый шум.
