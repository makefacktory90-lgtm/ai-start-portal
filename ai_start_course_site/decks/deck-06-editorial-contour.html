// НЕ АВТОПИЛОТ

AI может группировать сигналы. Важность определяет редакция.

Автоматизация мониторинга без редакционного правила быстро превращается в шум. Сначала источники, рубрики, критерии важности, риски. Потом уже автоматизация.

// PROMPT

Промпт для weekly digest

Собери редакционный weekly digest из списка сигналов.

Для каждого сигнала укажи:
1. Тема.
2. Источник и уровень доверия.
3. Почему это важно аудитории.
4. Возможный формат: новость, объяснение, видео, карточки, Telegram.
5. Риск: факт, контекст, юридический, репутационный.
6. Что должен решить редактор.

После таблицы предложи content grid на неделю:
канал / формат / owner / source / gate.
