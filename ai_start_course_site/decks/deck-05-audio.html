// ПОЗИЦИЯ

Разговорность — это не "добавь эмоций". Это другой синтаксис.

В устной версии короче фразы, яснее переходы, меньше канцелярита, больше дыхания. Но фактический слой остается тем же.

// PROMPT

Промпт для spoken script

Сделай spoken script по материалу.

Вход:
- Source Map.
- Канал: [эфир / подкаст / Telegram audio / voiceover].
- Длина: [60-90 секунд].
- Голос: спокойно, ясно, без рекламного пафоса.

Правила:
1. Не добавляй факты, которых нет в Source Map.
2. Сохрани смысл и масштаб события.
3. Пиши так, чтобы это можно было произнести вслух.
4. Отдельно отметь трудные имена, цифры и места для проверки.
5. Дай 2 варианта первого предложения.
