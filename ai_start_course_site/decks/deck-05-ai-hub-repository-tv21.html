02 · prompts

библиотека

обновления

// ДОМАШНЕЕ ЗАДАНИЕ

Два уровня работы с редакционным репозиторием.

base · Яндекс.Диск

Начать работать с репозиторием редакции на Яндекс.Диске: создать структуру, разнести материалы, базовые промпты и рабочие промпты по отдельным папкам.

advanced · GitHub

Начать работать с репозиторием в GitHub: перенести структуру, оформить README, зафиксировать правила обновления и роли владельцев.

update loop · Claude

Подключить репозиторий к Claude и настроить минимум ежемесячное обновление папки рабочих промптов на основе практики редакции.

Базовые промпты храним как стабильную основу. Рабочие промпты обновляем по результатам применения, правок редактора и повторяющихся задач.
